# OASM Fonts

This folder contains starter font sets for OASM shells:

- **Nerdy (Developer)**: Fira Code — monospaced font with programming ligatures.
- **Classical (Serif)**: EB Garamond — revival of Claude Garamont’s 16th century typeface.
- **Creative (Casual)**: Comic Neue — modernized casual script.

Update `fonts.json` to add new fonts or change defaults.
